App overview

A simple PLC learning app.

The app is built for practice on phone and tablet, not for real machine approval.

Learn the basics

Follow short lessons about PLC inputs, outputs, scan cycle, and common control ideas.

Practice ladder logic

Build small ladder examples and see how contacts, coils, timers, and interlocks behave.

Run safe simulations

Map tags to simulated devices and test the result without connecting to real equipment.
